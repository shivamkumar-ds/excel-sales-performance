# Dynamic Referencing & Advanced Lookup Techniques

---

# 1. INDIRECT Function

## Purpose
Returns the value of a reference constructed as text.

Instead of referring directly to a cell, it refers to the address stored in a cell.

---

## Syntax

```

=INDIRECT(ref_text, [a1])

```

- `ref_text` → text containing cell reference
- `[a1]` → TRUE (A1 style, default)  
           FALSE (R1C1 style)

---

## Basic Example

If C12 contains "G3":

```

=INDIRECT(C12)

```

Returns value in G3.

---

## R1C1 Example

If cell contains "R3C8":

```

=INDIRECT(A1, FALSE)

```

Returns value from Row 3 Column 8.

---

## Using INDIRECT with MATCH

Dynamic column reference example:

```

=INDIRECT("G"&MATCH(County, County_List, 0))

```

Used to:
- Retrieve agent name
- Retrieve contact number

---

## INDIRECT with Named Ranges

If month name is stored in C5:

```

=SUM(INDIRECT(C5))

```

Returns total sales for selected month.

---

## INDIRECT with Structured References

Dynamic SUMIFS using table name:

```

=SUMIFS(INDIRECT(Month),INDIRECT("Sales_"&Month&"[District]"),Selected_District)

```

Used when:
- Multiple similar tables
- Dynamic month selection

---

## Cascading Drop-Down List

Data Validation → List → Source:

```

=INDIRECT(C3)

```

Where C3 contains selected county  
Named ranges must match county names.

---

# 2. ADDRESS Function

## Purpose
Creates a cell reference using row and column numbers.

Often combined with INDIRECT.

---

## Syntax

```

=ADDRESS(row_num, column_num, [abs_num], [a1], [sheet])

```

- Default → absolute reference
- `abs_num` controls relative/absolute type
- `a1` → TRUE (A1 style), FALSE (R1C1)
- `sheet` → optional sheet name

---

## Basic Example

```

=ADDRESS(3,7)

```

Returns: $G$3

---

## Combine ADDRESS + INDIRECT

```

=INDIRECT(ADDRESS(row, column))

```

Returns value from constructed address.

---

## Dynamic Range Start

```

=ADDRESS(ROW(County_List),COLUMN(County_List))

```

Returns first cell of named range.

---

## Dynamic Range End

```

=ADDRESS(ROW(County_List) + ROWS(County_List) - 1,COLUMN(County_List))

```

Returns last cell of named range.

---

## Flexible Lookup Example

```

=INDIRECT(ADDRESS(ROW(County_List)+MATCH(Selected_County, County_List, 0) - 1,COLUMN(County_List)+1))

```

# 3. OFFSET Function

## Purpose
Returns reference a specified number of rows and columns away from a starting cell.

Can return:
- Single cell
- Range (using height and width)

---

## Syntax

```

=OFFSET(reference, rows, cols, [height], [width])

```

- `reference` → starting cell
- `rows` → move down (+) or up (-)
- `cols` → move right (+) or left (-)
- `height` → number of rows to return
- `width` → number of columns to return

---

## Basic Example

```

=OFFSET(A1,4,2)

```

Returns value 4 rows down, 2 columns right.

---

# Complex Lookup with OFFSET

Used when:
- 3-dimensional lookup
- Multiple conditions
- Table blocks

Example logic:

1. Find lender row (INDEX + MATCH)
2. Calculate row offset
3. Calculate column offset
4. Return value

Column offset example:

```

=MATCH(Loan_Amount, Loan_Thresholds, 1) + 1

```

Row offset example:

```

=ROUNDDOWN(Borrow_Percentage - 80.01, 0)

```

---

# OFFSET with Height & Width

Return multiple values:

```

=OFFSET(reference, row_offset, col_offset, 1, 3)

```

Returns 1 row, 3 columns.

If destination layout differs:

```

=TRANSPOSE(OFFSET(...))

```

Use Ctrl + Shift + Enter (if required).

---

# 4. Combo Box (Form Control)

Developer → Insert → Combo Box

Steps:
1. Set Input Range
2. Set Cell Link
3. Cell link returns position number (1, 2, 3...)

Use that number inside OFFSET to pull correct row.

---

# 5. Dynamic Named Ranges (Using OFFSET)

Charts cannot directly use OFFSET in cells.  
Must create named ranges.

Example:

```

=OFFSET(
$C$20,
0,
0,
COUNTIFS($C$20:$C$329,">0")
)

```

Creates dynamic vertical range.

---

# 6. Key

- INDIRECT → converts text to reference
- ADDRESS → builds reference
- OFFSET → shifts reference
- INDEX + MATCH → locate base reference
- Dynamic ranges → auto-expanding charts
- Structured references → table-based formulas
- Combo box → user-driven input control
